¿Qué es una Signal?

Una Signal es básicamente una caja reactiva que guarda un valor.

  • Cuando el valor cambia, Angular detecta automáticamente el cambio.
  • Los componentes o plantillas que usan esa signal se vuelven a renderizar solo cuando es necesario, sin tener que disparar ChangeDetection completo. 
    • component.ts
    n1 = signal(0);
    n2 = signal(0);
    result = computed(() => this.n1() * this.n2());

    componet.html
    <input
      [ngModel]="n1()"
      (ngModelChange)="n1.set($event)"
    />"
    <input
      [ngModel]="n2()"
      (ngModelChange)="n2.set($event)"
